О Бирже и источнике данных

Данные берутся из официального ISS MOEX API (Informational & Statistical Server), который предоставляет онлайн‑доступ к рынкам Московской биржи.

  • Акции: движок stock, рынок shares, борд TQBR
  • Фьючерсы: движок futures, рынок forts, борд RFUD

Тепловая карта рассчитывает изменение цены в процентах: (LAST - PREVPRICE) / PREVPRICE * 100.
Для фьючерсов используется PREVSETTLEPRICE, если PREVPRICE недоступен.

Сайт работает локально, без серверной БД; обновление происходит при каждом открытии страницы.
